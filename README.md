# Auto-Retardation-System
Prototype model to demonstrate the radar system for cruise control in cars

Sensor used: SR-04(Ultrasonic Sensor);
Microcontroller: Atmega 328p;
IDE: Arduino;
Other Components: HC-04 (Bluetooth modue);
                  L293D (Motor Driver);
